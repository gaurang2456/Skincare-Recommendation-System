# AI-Powered Skincare Recommendation System

A portfolio-ready machine learning application built with Python and Streamlit. It delivers personalised skincare product recommendations and daily routines based on your age, skin type, skin concerns, and ingredient preferences — all powered by a locally trained Random Forest model.

---

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Generate the synthetic training dataset

```bash
python ml/generate_dataset.py
```

This produces `data/skincare_dataset.csv` (≥ 1,000 records) covering five skin types and ten concern categories.

### 3. Train the ML model

```bash
python ml/train_model.py
```

This trains a `MultiOutputClassifier(RandomForestClassifier)` on the generated dataset, serialises the model artifact to `models/skincare_model.pkl`, and writes evaluation metrics to `data/model_metrics.json`.

### 4. Launch the application

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

---

## Running Tests

```bash
pytest tests/ -v
```

All unit tests are located in `tests/` and cover scoring logic, routine generation, recommendation history, ingredient filtering, and PDF report generation.

## Application Pages

| # | Page | Purpose |
|---|------|---------|
| 1 | **Home** (`app.py`) | Introduction to the app with a prominent "Start Assessment" button that navigates to the assessment form. |
| 2 | **Assessment** (`pages/1_Assessment.py`) | Form collecting age, skin type, concerns with per-concern severity sliders, and optional ingredient exclusions. Validates inputs and redirects to Recommendations on success. |
| 3 | **Recommendations** (`pages/2_Recommendations.py`) | Displays the Skin Health Score, four personalised product recommendations with confidence scores, a morning and night skincare routine, and a PDF download button. |
| 4 | **Recommendation History** (`pages/3_History.py`) | Shows all past assessments and their recommended products in reverse chronological order, persisted locally as a CSV file. |
| 5 | **ML Insights** (`pages/4_ML_Insights.py`) | Developer-focused page showing dataset overview, model evaluation metrics (accuracy, precision, recall, F1), feature importance chart, confusion matrix, and skin-type class distribution. |
